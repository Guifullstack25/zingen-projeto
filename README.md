# 🎵 Zingen

Landing page de um aplicativo fictício de karaokê chamado **Zingen**.

O projeto foi desenvolvido com foco em **HTML e CSS**, simulando a página inicial de divulgação de um app para cantar músicas utilizando inteligência artificial.

---

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* Flexbox
* Responsividade (Mobile First)

---

## 📱 Funcionalidades da página

* Header com navegação
* Hero section com chamada principal
* Botões de ação (Ver planos / Baixar agora)
* Ilustração de smartphones
* Background com gradiente em SVG
* Layout responsivo
